PANDUAN PENGGUNAAN UNTUK PENGGUNA MAC DAN LINUX
===============================================

Bot ini sekarang mendukung macOS dan Linux. Karena file .exe hanya untuk Windows, pengguna Mac/Linux harus menjalankan kode sumber (source code) secara langsung menggunakan Python.

LANGKAH-LANGKAH INSTALASI:
--------------------------

1. Pastikan Python 3 sudah terinstall di komputer Anda.
   - Buka Terminal dan ketik: `python3 --version`
   - Jika belum ada, download dan install dari: https://www.python.org/downloads/

2. Buka Terminal dan masuk ke folder ini.
   Contoh: `cd Downloads/source_code_for_mac_linux`

3. Install library yang dibutuhkan dengan perintah:
   
   pip3 install -r requirements.txt

   (Jika gagal, coba gunakan `pip` saja: `pip install -r requirements.txt`)

4. Jalankan bot dengan perintah:

   python3 autofishbot.py

   (Atau: `python autofishbot.py`)

5. Masukkan password lisensi ketika diminta (sama seperti versi Windows).

CATATAN:
- Pastikan koneksi internet stabil.
- Untuk menghentikan bot, tekan Ctrl+C di terminal.
